# E-Commerce SQL Data Analysis Project

## Overview

This project focuses on analyzing an E-Commerce database using SQL to solve real-world business problems and generate business insights.

The project includes structured SQL queries covering customer analysis, sales analysis, product insights, inventory tracking, and reporting scenarios.
